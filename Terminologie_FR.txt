## Dictionnaire de la terminologie WordPress ##

Ce document est destiné à l'usage des traducteurs du Codex

***

Checkbox = case à cocher

Dashboard = tableau de bord

Panel (ex: In the Settings → Permalinks panel ) = page de configuration (?)

Permalink = permalien

Pretty Permalinks = permaliens simplifiés
( note: dans Prestashop, c’est URL simplifiée / ou "Friendly URLs")

Tags (structure tags) = marqueurs de structure

Template = modèle (changé par Max)
